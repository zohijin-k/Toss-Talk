# 💬 Toss-Talk

AI와의 대화를 통해 감정을 표현하고, 시각적 상호작용과 미니게임을 통해 감정 해소를 돕는 감정 케어 서비스

🏆 **2025 교내 해커톤 우수상**

---

## 📌 Overview

Toss-Talk은 사용자가 자신의 감정을 편하게 털어놓고 해소할 수 있도록 설계된 AI 기반 감정 케어 서비스입니다.

사용자는 AI 챗봇과 대화를 나눈 뒤, 서비스 이용 전후의 감정 상태를 비교하여 자신의 감정 변화를 직관적으로 확인할 수 있습니다.

또한 감정을 종이에 적어 구긴 뒤 쓰레기통에 버리는 인터랙션과 간단한 미니게임을 통해 감정을 해소하는 경험을 제공합니다.

---

## ✨ Key Features

### 🤖 AI Chat
AI와 자유롭게 대화하며 자신의 감정을 표현할 수 있습니다.

### 😊 Emotion Change Visualization
서비스 이용 전과 후의 감정 상태를 비교하여 보여줍니다.

예시:  
힘듦 → 괜찮아짐

### 📝 Emotion Disposal Interaction
감정을 종이에 적고, 종이를 구겨 쓰레기통으로 드래그하여 버리는 감정 해소 인터랙션을 제공합니다.

### 🌱 Garden Mini Game
상담 후 정원 가꾸기 게임으로 참여도를 높입니다.

### 🥚 Stress Relief Game
단순한 계란 깨기 게임을 통해 즉각적인 스트레스 해소 경험을 제공합니다.
